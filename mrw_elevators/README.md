# mrw_elevators
 
🔥 Features 
Supports ESX, QB 
Supported creation of all custom elevators for jobs 
Customizable notifications ( Default settings for OX - ox_inventory / ox_lib ) 

🛠️ Dependencies 
ox_lib 
ox_inventory 
📲 Installation 
Download latest release or source code 
Extract the .zip file 
Copy the folder to your server resources folder 
Add ensure mrw_preloading to your server.cfg 
Restart the server
